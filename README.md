# Xiaomi M365 Display 
-Added hibernate mode under KERS config.
-The arduino nano now monitors the arduino pro via the pin 12.

# Products Used  
Arduino Pro Mini    
I2C OLED 0.96" Screen    
FTD1232 Usb Programmer   
3d Printed Bracket  
1N4148 Diode  
120ohm Resistor  
Arduino Nano

Estimated price is around 15$.
